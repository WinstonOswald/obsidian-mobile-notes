# Obsidian Note Creator - Mobile

Create hierarchical note structures on your phone for Obsidian vault.

## Usage

Visit the app: [https://winstocalude.github.io/obsidian-mobile-notes](https://winstocalude.github.io/obsidian-mobile-notes)

1. Type your note hierarchy (use spaces for indentation)
2. Tap "Parse Structure"
3. Tap each note to add content and aliases
4. Tap "Export" to download JSON
5. Import the JSON on your desktop app

## Features

- ✅ Works on any mobile browser
- ✅ Touch-optimized interface
- ✅ Auto-saves your work
- ✅ Export to JSON for desktop
- ✅ Offline support
